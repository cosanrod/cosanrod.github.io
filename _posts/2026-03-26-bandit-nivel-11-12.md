### 🧠 Explicación

#### 🔐 ¿Qué es ROT13?

Es un tipo de cifrado clásico llamado **cifrado César**:

* Cada letra se desplaza **13 posiciones** en el alfabeto
* Es reversible aplicando el mismo proceso

#### 🔤 Ejemplo:

```bash id="u3x7qp"
a → n  
b → o  
c → p  
```

Entonces:

```bash id="h5k2mz"
hola → ubyn
```

---

### ⚙️ Uso de `tr`

El comando:

```bash id="q7v4dn"
tr 'a-zA-Z' 'n-za-mN-ZA-M'
```

Hace una sustitución de caracteres:

* `a-zA-Z` → rango original
* `n-za-mN-ZA-M` → letras rotadas 13 posiciones
