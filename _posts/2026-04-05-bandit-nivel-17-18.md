### 🧠 Explicación

El comando `diff` muestra las diferencias entre dos archivos línea por línea.

#### 🔍 Interpretación del resultado:

```bash id="m2c3qy"

```

👉 La línea 42 del primer archivo es diferente a la línea 42 del segundo.

```bash id="x7k4rb"

```

👉 `<` corresponde al **primer archivo** (`passwords.new`)

```bash id="z3n5lw"

```

👉 `>` corresponde al **segundo archivo** (`passwords.old`)
