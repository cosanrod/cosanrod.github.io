### 🎯 Objetivo

Obtener la contraseña del siguiente nivel.

Existe un binario SUID llamado:


1. Se conecta a `localhost` en un puerto que tú especifiques
2. Lee una línea de texto
3. La compara con la contraseña actual
4. Si coincide, devuelve la contraseña del siguiente nivel

### 🧠 Explicación

Este nivel funciona como un **cliente-servidor invertido**:

* `suconnect` → actúa como cliente
* `nc` → actúa como servidor

---

### ⚙️ Desglose del comando

```bash id="q7v4dn"
echo "password" | nc -l -p 1234 &
```

| Componente | Función                  |
| ---------- | ------------------------ |
| `echo`     | Envía la contraseña      |
| `nc -l`    | Escucha conexiones       |
| `-p 1234`  | Puerto local             |
| `&`        | Ejecuta en segundo plano |

---

### 🔄 Flujo completo

1. Abres un puerto con `nc`
2. `suconnect` se conecta
3. Recibe la contraseña
4. Valida como `bandit21` (por SUID)
5. Te devuelve la nueva contraseña
